# Placeholder Icon File

Ce fichier est un placeholder pour l'icône de l'application.

## Instructions

1. **Créer votre icône**: Créez une icône au format ICO avec les résolutions suivantes:
   - 16x16
   - 32x32
   - 48x48
   - 64x64
   - 128x128
   - 256x256

2. **Remplacer ce fichier**: Placez votre fichier `icon.ico` dans ce dossier

3. **Outils recommandés**: 
   - IcoFX (Windows)
   - GIMP + ICO plugin
   - Convertisseurs en ligne (attention à la qualité)

## Format

- **Type**: ICO multi-résolution
- **Profondeur de couleur**: 32-bit avec canal alpha (transparence)
- **Taille max**: < 1 MB

## Alternative

Si vous n'avez pas d'icône personnalisée, l'application utilisera l'icône par défaut d'Electron lors du build.

Pour créer une icône rapidement à partir d'une image PNG:
- Utilisez https://icoconvert.com/
- Sélectionnez toutes les résolutions
- Téléchargez et placez ici
